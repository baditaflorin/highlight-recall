# Code Of Conduct

## Our Pledge

We pledge to make participation in this project respectful and harassment-free.

## Expected Behavior

- Be kind and specific.
- Assume good intent while still naming real problems.
- Respect privacy, especially around personal reading data and exported libraries.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private user data.
- Introducing tracking without an accepted ADR.

## Enforcement

Maintainers may remove comments, issues, or contributions that violate this code.
